# 🔍 Account Checker Bot

Бот для проверки информации об аккаунтах Telegram по ID.  
Показывает приблизительную дату создания, датацентр, возраст и категорию аккаунта.

---

## ⚡️ Возможности

- 🔍 Анализ аккаунта по ID
- 📅 Приблизительная дата создания (по диапазону ID)
- 🖥 Определение датацентра (DC)
- 🏷 Категория аккаунта (Новичок / Обычный / Опытный / Старожил)
- ⏰ Лимит 3 запроса в час на пользователя
- 🛡 Админ-панель с уведомлениями и баном

---

## 🚀 Деплой на Railway

### 1. Создай репозиторий на GitHub
- Загрузи файлы: `bot.py`, `requirements.txt`, `Procfile`

### 2. Зайди на [railway.app](https://railway.app)
- `New Project` → `Deploy from GitHub repo`
- Выбери свой репозиторий

### 3. Добавь переменную окружения
```
BOT_TOKEN = твой_токен_от_BotFather
```

### 4. Готово! 🎉
Бот запустится автоматически.

---

## 🛠 Команды

### Для всех:
| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота |

### Только для админа:
| Команда | Описание |
|---------|----------|
| `/stats` | Статистика пользователей |
| `/ban <ID>` | Забанить пользователя |
| `/unban <ID>` | Разбанить пользователя |

---

## 👑 Админ-панель

Когда новый пользователь пишет боту — тебе приходит уведомление:

```
🆕 Новый пользователь!

👤 Имя: Иван
🔗 Username: @ivan
🆔 ID: 123456789
🕐 Время: 22.03.2026 10:48

[🚫 Забанить]  [✅ Разбанить]
```

Нажми кнопку — пользователь сразу получит уведомление о бане/разбане.
